# School Management System

A role-based School Management System built using PHP, MySQL, and Bootstrap.

## Features
- Admin & Teacher login
- Multiple classes per teacher
- Attendance management
- Attendance reports
- Responsive dashboard UI

## Tech Stack
- PHP
- MySQL
- Bootstrap
- HTML, CSS, JavaScript

## Setup Instructions
1. Clone the repository
2. Import the database (.sql file)
3. Rename `config/db.example.php` to `db.php`
4. Update database credentials
5. Run on XAMPP / Live server
